HTML Tags Reference

This table includes a list of commonly used HTML tags along with their description and an example of usage.

Tag Example Description
<html> <html>...</html> Defines the root of an HTML document.
<head> <head>...</head> Contains metadata for the document.
<body> <body>...</body> Defines the body of the document.
<header> <header>...</header> Defines a header section.
<footer> <footer>...</footer> Defines a footer section.
<h1>-<h6> <h1>Heading</h1> Defines headings from level 1 to 6.
<p> <p>This is a paragraph.</p> Defines a paragraph.
<a> <a href="https://example.com">Link</a> Defines a hyperlink.
<img> <img src="image.jpg" alt="Image"> Defines an image.
<table> <table>...</table> Defines a table.
<tr> <tr>...</tr> Defines a table row.
<td> <td>Data</td> Defines a table cell.
<th> <th>Header</th> Defines a header cell in a table.
<ul> <ul>...</ul> Defines an unordered list.
<ol> <ol>...</ol> Defines an ordered list.
<li> <li>Item</li> Defines a list item.
<div> <div>...</div> Defines a division or section.
<span> <span>Text</span> Defines an inline container.
<form> <form>...</form> Defines a form for user input.
<input> <input type="text"> Defines an input field.
<button> <button>Click</button> Defines a clickable button.
<select> <select>...</select> Defines a dropdown list.
<option> <option>Choice</option> Defines an option in a dropdown.
<textarea> <textarea>...</textarea> Defines a multiline text input.
<nav> <nav>...</nav> Defines navigation links.
<section> <section>...</section> Defines a section.
<article> <article>...</article> Defines an article.
<aside> <aside>...</aside> Defines content aside from the main content.
<main> <main>...</main> Defines the main content of the document.
<audio> <audio controls>...</audio> Defines audio content.
<video> <video controls>...</video> Defines video content.
<iframe> <iframe src="https://example.com"></iframe> Defines an inline frame.
<strong> <strong>Text</strong> Defines strong importance.
<em> <em>Text</em> Defines emphasized text.
<mark> <mark>Text</mark> Defines highlighted text.
<code> <code>Code</code> Defines computer code.
<pre> <pre>...</pre> Defines preformatted text.
<blockquote> <blockquote>...</blockquote> Defines a block quotation.
<hr> <hr> Defines a horizontal rule.
<br> <br> Inserts a line break.
<sup> <sup>Text</sup> Defines superscript text.
<sub> <sub>Text</sub> Defines subscript text.

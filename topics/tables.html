HTML Table Tags

HTML tables are used to represent tabular data in rows and columns. Tables allow you to organize data in a structured format, making it easier to read and understand.

1. Basic Table Structure

The basic structure of an HTML table involves the following tags:

  • <table>: Defines the table.
  • <tr>: Defines a table row.
  • <th>: Defines a header cell.
  • <td>: Defines a data cell.
  • <thead>: Groups the header content in the table.
  • <tbody>: Groups the body content in the table.

Syntax:

<table>
  <thead>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
      <td>Data 2</td>
    </tr>
  </tbody>
</table>

Example:

Rendered Output:

Header 1 Header 2
Data 1 Data 2

2. Table with Multiple Rows and Columns

Tables can have multiple rows and columns to display more data.

Syntax:

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>City</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John</td>
      <td>30</td>
      <td>New York</td>
    </tr>
    <tr>
      <td>Alice</td>
      <td>25</td>
      <td>London</td>
    </tr>
  </tbody>
</table>

Example:

Rendered Output:

Name Age City
John 30 New York
Alice 25 London

3. Table with Column Span (<colspan>)

You can merge cells in a row by using the <colspan> attribute.

Syntax:

<table>
  <thead>
    <tr>
      <th colspan="2">Merged Header</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
      <td>Data 2</td>
    </tr>
  </tbody>
</table>

Example:

Rendered Output:

Merged Header
Data 1 Data 2

4. Table with Row Span (<rowspan>)

You can merge cells in a column by using the <rowspan> attribute.

Syntax:

<table>
  <thead>
    <tr>
      <th rowspan="2">Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
    </tr>
  </tbody>
</table>

Example:

Rendered Output:

Header 1 Header 2
Data 1

5. Table with Caption (<caption>)

A <caption> tag can be used to add a title to the table.

Syntax:

<table>
  <caption>Table Title</caption>
  <thead>
    <tr>
      <th>Column 1</th>
      <th>Column 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
      <td>Data 2</td>
    </tr>
  </tbody>
</table>

Example:

Rendered Output:

Table Title
Column 1 Column 2
Data 1 Data 2

6. Summary Table

Here is a summary of the table tags:

Tag Description
<table> Defines the entire table structure.
<thead> Groups the header content in the table.
<tbody> Groups the body content in the table.
<tr> Defines a row inside the table.
<th> Defines a header cell in the table.
<td> Defines a data cell in the table.
<caption> Adds a title or caption to the table.
<colspan> Spans multiple columns in a header or data cell.
<rowspan> Spans multiple rows in a header or data cell.

Product Inventory

This table represents the current product inventory, showing the product name, category, price, quantity, and total value.

Syntax:

<table>
  <thead>
    <tr>
      <th>Product Name</th>
      <th>Category</th>
      <th>Price</th>
      <th>Quantity</th>
      <th>Total Value</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>Electronics</td>
      <td>$800</td>
      <td>15</td>
      <td>$12000</td>
    </tr>
  </tbody>
</table>

Example:

Product Name Category Price Quantity Total Value
Laptop Electronics $800 15 $12000
Smartphone Electronics $500 30 $15000
Headphones Accessories $100 50 $5000

Order Management

This table displays the list of orders with the order number, customer name, order date, and status.

Syntax:

<table>
  <thead>
    <tr>
      <th>Order Number</th>
      <th>Customer Name</th>
      <th>Order Date</th>
      <th>Status</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ORD001</td>
      <td>John Doe</td>
      <td>2024-12-01</td>
      <td>Shipped</td>
    </tr>
  </tbody>
</table>

Example:

Order Number Customer Name Order Date Status
ORD001 John Doe 2024-12-01 Shipped
ORD002 Alice Smith 2024-12-02 Processing
ORD003 Bob Johnson 2024-12-03 Pending

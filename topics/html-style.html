HTML Styles and Colors

HTML provides various styling and coloring options to enhance the visual appearance of the content.

1. HTML Styles

CSS is used to style HTML elements. You can apply styles using various methods such as inline styles, internal styles, and external stylesheets.

Inline Style:

<div style="color: blue; font-size: 20px;">This is a styled div.</div>

Example:

This is a styled div with inline styles.

Internal Style:

<style>
  p {
    color: green;
    font-size: 18px;
  }
</style>

Example:

This paragraph is styled using internal CSS.

External Style:

External styles are defined in a separate CSS file and linked to the HTML document.

<link rel="stylesheet" href="styles.css">

styles.css

p {
 color: red;
 font-size: 30px;
}

Example:

This paragraph is styled using internal CSS.

2. HTML Colors

HTML supports a wide range of colors that can be applied to text, backgrounds, borders, and other elements.

Color Names:

<p style="color: red;">This is red text.</p>

Example:

This is red text.

Hexadecimal Colors:

<p style="color: #00FF00;">This is green text using hexadecimal color.</p>

Example:

This is green text using hexadecimal color.

RGB Colors:

<p style="color: rgb(255, 0, 0);">This is red text using RGB color.</p>

Example:

This is red text using RGB color.

RGBA Colors (with Transparency):

<p style="color: rgba(255, 0, 0, 0.5);">This is semi-transparent red text.</p>

Example:

This is semi-transparent red text.

3. Summary Table

Here is a summary of styling and color options:

Concept Method Description
Inline Style style attribute Styles applied directly to HTML elements.
Internal Style <style> tag Styles defined within the HTML document.
External Style <link> tag Styles defined in an external CSS file.
Color Names Color keywords (e.g., red) Predefined color names in CSS.
Hexadecimal Colors #RRGGBB Colors specified using hex codes.
RGB Colors rgb(r, g, b) Colors defined using red, green, blue values.
RGBA Colors rgba(r, g, b, a) RGB colors with transparency (alpha channel).

Basics of HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure of a webpage and allows the inclusion of text, images, links, and more.

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages and web applications. It works in conjunction with CSS for styling and JavaScript for functionality.

2. Creating a "Hello World" HTML File

Using Notepad

  1. Open Notepad.
  2. Type the following HTML code:
    3. 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello World</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
  3. Save the file with a .html extension, e.g., hello.html.
  4. Open the saved file in any web browser to view the output.

Using Visual Studio Code (VS Code)

  1. Open VS Code and create a new file.
  2. Type the same HTML code as above.
  3. Save the file with a .html extension, e.g., hello.html.
  4. Right-click the file and select Open with Live Server (if the Live Server extension is installed).
  5. The browser will display the "Hello World" output.

3. HTML Structure

A basic HTML document includes the following structure:


<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Title of the Document</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Explanation of components:

  • <!DOCTYPE html>: Declares the document type as HTML5.
  • <html>: The root element of the HTML document.
  • <head>: Contains metadata, such as the title and character encoding.
  • <body>: Contains the visible content of the webpage.

4. Common Tags

Below are some commonly used HTML tags with examples:

  • <h1> to <h6>: Heading tags, where <h1> is the largest and <h6> is the smallest. 
    
<h1>Heading 1</h1>
<h2>Heading 2</h2>
  • <p>: Paragraph tag. 
    
<p>This is a paragraph.</p>
  • <a>: Anchor tag for hyperlinks. 
    
<a href="https://www.example.com">Visit Example</a>
  • <img>: Image tag. 
    
<img src="image.jpg" alt="Description of Image">
  • <div>: Division tag for grouping content. 
    
<div>This is a division.</div>
  • <span>: Inline container for styling or grouping. 
    
<span style="color: red;">This is red text.</span>

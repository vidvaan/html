HTML Lists

HTML provides various types of lists to organize content effectively. Lists help in presenting information in a structured and hierarchical manner.

1. Ordered Lists (<ol>)

An ordered list displays items in a sequential order, typically numbered.

Syntax:

<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

Example:

Rendered Output:

  1. First Item
  2. Second Item
  3. Third Item

2. Unordered Lists (<ul>)

An unordered list displays items with bullet points, without any specific sequence.

Syntax:

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>

Example:

Rendered Output:

  • Apple
  • Banana
  • Cherry

3. Definition Lists (<dl>, <dt>, <dd>)

A definition list displays terms and their associated descriptions, useful for glossaries or dictionaries.

Syntax:

<dl>
  <dt>HTML</dt>
  <dd>A markup language for creating web pages.</dd>
  <dt>CSS</dt>
  <dd>A language for styling web pages.</dd>
</dl>

Example:

Rendered Output:

HTML
A markup language for creating web pages.
CSS
A language for styling web pages.

4. Nested Lists

Lists can be nested to represent hierarchical or categorized data.

Syntax:

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ol>
      <li>Carrot</li>
      <li>Broccoli</li>
    </ol>
  </li>
</ul>

Example:

Rendered Output:

  • Fruits
    • Apple
    • Banana
  • Vegetables
    1. Carrot
    2. Broccoli

5. Summary Table

Here is a summary of the list types with their characteristics:

List Type Tag Description
Ordered List <ol> Displays items in a sequential order with numbers or letters.
Unordered List <ul> Displays items with bullet points.
Definition List <dl>, <dt>, <dd> Displays terms and their definitions.
Nested List Combination of <ol>, <ul>, <dl> Lists within lists for hierarchical data.

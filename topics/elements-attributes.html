HTML Elements and Attributes

1. Understanding HTML Elements

An HTML element consists of a start tag, content, and an end tag. Some elements, called empty elements, don’t have content or an end tag.

Example: Standard Elements


<p>This is a paragraph.</p>
<h1>This is a heading.</h1>

Example: Empty Elements

Empty elements don’t have content or closing tags.


<br> 
<img src="image.jpg" alt="Description of the image">

2. HTML Attributes

Attributes provide additional information about an element. They are always included in the opening tag and usually have a name and a value.

Common Attributes

  • id: Specifies a unique identifier for an element.
  • class: Specifies a class name for one or more elements (used in CSS/JavaScript).
  • src: Specifies the source of an image, video, or other external content.
  • href: Specifies the URL of a link.
  • alt: Provides alternative text for an image.
  • title: Specifies additional information about an element (displayed as a tooltip).

Examples of Attributes


<h1 id="main-title">Welcome to HTML</h1>
<div class="container">This is a container.</div>
<img src="logo.png" alt="Company Logo">
<a href="https://www.example.com" title="Visit Example">Go to Example</a>

3. Global Attributes

Global attributes can be used on any HTML element and provide additional functionality.

  • style: Applies inline CSS styles.
  • data-*: Stores custom data attributes.
  • accesskey: Specifies a shortcut key to activate or focus an element.
  • tabindex: Specifies the tab order of an element.

Examples of Global Attributes


<p style="color: blue; font-size: 18px;">This text is styled.</p>
<div data-user-id="12345">User ID: 12345</div>
<a href="#section1" accesskey="s">Go to Section 1</a>
<input type="text" tabindex="1">

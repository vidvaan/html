Text Formatting in HTML

HTML provides various tags for formatting text, making it bold, italicized, underlined, highlighted, and more. Proper use of these tags improves readability and accessibility.

1. Basic Text Formatting Tags

Below are commonly used text formatting tags with their descriptions and examples:

Tag Description Example
<b> Makes text bold (not semantically important). <b>Bold Text</b> → Bold Text
<strong> Indicates strong importance and makes text bold. <strong>Strong Text</strong> → Strong Text
<i> Makes text italic (not semantically important). <i>Italic Text</i> → Italic Text
<em> Indicates emphasis and makes text italic. <em>Emphasized Text</em> → Emphasized Text
<u> Underlines text (use sparingly as it's not semantic). <u>Underlined Text</u> → Underlined Text
<mark> Highlights text (useful for marking important points). <mark>Highlighted Text</mark> → Highlighted Text
<small> Makes text smaller. <small>Small Text</small> → Small Text
<del> Indicates deleted or removed text (strikethrough). <del>Deleted Text</del> → Deleted Text
<ins> Indicates inserted or added text (usually underlined). <ins>Inserted Text</ins> → Inserted Text

2. Semantic Importance: vs.

The <strong> tag emphasizes text with semantic meaning, indicating that the text is of greater importance. In contrast, the <b> tag only applies bold styling without any semantic meaning.

Example:


<p>This is an example of <b>bold</b> and <strong>strong</strong> text.</p>

Rendered Output:

This is an example of bold and strong text.

3. Combining Tags

You can combine tags for more complex formatting.

Example:


<p>This is <b>bold</b>, <i>italicized</i>, and <mark>highlighted</mark> text.</p>

Rendered Output:

This is bold, italicized, and highlighted text.

4. Usage Tips

  • Use <strong> and <em> for semantic emphasis to improve accessibility and SEO.
  • Avoid overusing <u> as it may confuse users who associate underlines with links.
  • Use <mark> to highlight important points or search results.
